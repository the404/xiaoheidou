云海谷俱乐部为杰克·尼克劳斯父子联袂担纲设计的锦标赛球场。湖景球场为杰克•尼克劳斯签名设计，几乎每一个球洞都可以一览阳宗海全景。杰克•尼克劳斯的设计充分利用项目地的壮美风景、连绵起伏的地势和自然植被，为球手打造一个中国最为激动人心的高尔夫体验地。

山景球场为杰克•尼克劳斯二世签名设计，位于项目地海拔较高处，将为您带来令人难忘且富有挑战性的高尔夫体验。不论差点高低，您都将享受到该度假球场的壮丽景色、浑然天成的沙坑、具备可打性的宽阔球道落球区和果岭边切杆区。

36洞的锦标赛球场充分利用项目地连绵起伏的地势，打球时不仅可以充分享受高尔夫运动的挑战性，还可以欣赏壮美的山海景观。湖景球场几乎每一洞都能欣赏阳宗海全景，山景球场则需要在大气磅礴的山谷间挥杆。
