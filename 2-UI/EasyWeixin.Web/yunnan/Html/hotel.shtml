群樱荟坐落于体育公园核心地带，由曾荣膺全球最佳设计图、2008年度最佳会所室内设计企业的美国MAI公司精心设计。项目由会所、酒店、公寓三大部分组成。

群樱荟公寓有着无可比拟的景观资源，南侧直面阳宗海碧波蓝天，四周则是永久绿地和森林环绕，西面的花谷苍峦历历在目，于满目苍翠之间领悟山水的妙趣。独具匠心的超低密度三层建筑形态，毗邻群樱荟会所，居住空间幽静私密，交通、餐饮、娱乐设施便捷。在阳宗海之滨，于静逸之中品位时光的美好。

群樱荟公寓分套房和标间两大类型产品，面积区间为50-120㎡，主力户型为标间，价格区间为70-100万元/套。公寓采用五星级酒店标准精装，深圳华侨城洲际大酒店、北京万豪酒店、三亚丽思卡尔顿酒店等五星酒店御用家具供应商量身定制精品家私，国际一线品牌卫浴设备，享受拎包入住的无虞之旅。
