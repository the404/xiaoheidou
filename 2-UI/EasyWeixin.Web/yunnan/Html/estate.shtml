天麓 馨香园

天麓山海大宅，传承中国十大豪宅品牌、深圳东部华侨城天麓的纯正血统。以"生态大于天"的原则为核心理念，尽最大可能去实现建筑与原始环境的和谐共生。2011年，天麓一区实现开盘一小时即宣告售罄，创造了当年昆明地产界的奇迹。

天麓二区，坐拥云南华侨城项目制高点，西瞰生态花谷，南望阳宗海、群樱荟，私享超星级配套，占据最佳观景视野。二区占地12.72万平米，拥有0.24超低容积率，绿地率更是高达56%，为您带来近乎天然的舒适享受。天麓二区涵括200-251平米独栋别墅，以坡地双首层设计，实现格局舒适、空间开阔、居住私密的尊崇体验。独创吊脚楼空间形态，极大增加可使用的空间面积，并且配以地下室、星空露台附赠，为您的尊贵体验锦上添花。天麓二区采用天然气、太阳能及空气源热泵双系统供热水、地暖使用，为您带来舒适便捷的养生度假生活。
