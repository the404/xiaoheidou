• 尖叫大水环

    来找刺激？想玩心跳？还要夏日里的冰爽？这里就是你的最爱，20米的落差、瞬间加速以及转弯、滑行、跃起、入水等花样动作，让你尖叫响不停。

  • 疯狂麻花

    塔台高度近15米的6人水上赛道同场竞技，每个人头朝下、脚朝上由一条单独的滑道顺着急流往下俯冲，整个过程中仅能握住一个特制的滑水垫，人似乎瞬间就变成一辆水上跑车，那种刺激可能只有从悬崖上往瀑布里栽过的人才感受过！

  • 狂欢水寨

    快来参加水寨狂欢节吧，泼水、抢水还有水枪扫射，有什么花招，尽管使出来。一起加入水枪大战吧，与敌人激战一场。

  • 巨人冒险筏

    16米高的4人圆阀大滑道，游客在一个半圆型的管道中顺势滑行，除了身下的水流外，再没有什么可以凭借的外力了，置身其中宛如水中腾云。

  • 加勒比漂流

    高速！俯冲！旋转！还有动人心魄的急速转弯！淋漓尽致的快感顿时传遍全身。全方位释放出水上娱乐魅力，大声呐喊，享受水上极限快感吧！

  • 飓风大喇叭

    “飓风大喇叭”体积庞大，线条圆润，起点处有20多米高。当游客坐在特别设计的皮划艇里拐三个弯猛地垂直下落时，强大的水流伴随着自由落体的惯性，像被龙卷风吸进另一个空间，新鲜惊险刺激，有趣的是，待冲出“飓风”中心，惊魂未定的你才发现这段惊险体验只经历了10秒钟，却让你永生难忘。塔台高 21米，滑道长 198 米

  • 旋风大浪

    从21米的高度自由下落，在长达145米的滑道中翻腾跳跃，最后瞬间旋滑落到水中，这场颠簸之旅，让您从开始到结束秒秒惊险激情连连！

  • 加勒比海滩

    中国最大的室内外近5000平方米的造浪池。十多种创意造浪模式，叠浪、涌浪、钻石浪等，既可以享受风和日丽的浪漫风情，也能体验狂风怒号的汹涌澎湃！阳光、沙滩与浓郁的热带雨林相映成趣，呈现一派加勒比海滨风情，一切是那么轻松、愉快、休闲、欢乐，构成优美画卷。四季精彩欢乐，给您带来世界级海浪的万般魅力！

  • 漂流河

    在宽2米长190米的室外漂流河中，游客可以躺在特别设计的浮圈上沿河漂流，感受飞流而下的水花四溅，放松身心。此外，河中还会提供一段独特的急漂区域，峰回路转的急速转弯带一定能给你一个惊喜！
