全价票

身高1.5米（含1.5米）以上的儿童及成人。

优惠票

￥100/人

1、身高1.2米（含1.2米）至1.5米（不含1.5米）的儿童；

2、65周岁（含65周岁）至69周岁（含69周岁）的长者（凭第二代有效身份证购票入园）；

3、伤残人士持二代《残疾人证》、军残人士持《残疾军人证》可够优惠票入园，盲人、智障、生活不能自理的残疾人须具备安全监护能力的购票成人陪同入园。

夜场票

￥80/人

1、身高1.2米（含1.2米）以上的儿童及成人。

2、夜场票不设优惠票。

免票

￥0/人

1、身高1.2米(不含1.2米)以下儿童（每名购票成人限带一名低于1.2米的儿童）。

2、70周岁及以上长者，凭本人二代身份证入园。

小贴士 / TIPS

  • 1.时间：2014年11月17日—2015年3月15日。

  • 2.游客持票入园前，须认真阅读并承诺遵守检票口处《游客入园须知》。

  • 3.游客每人限享受一种优惠政策，拥有享受多种折扣资格的，按最低折扣计算。

  • 4.全价票有效期为7天，优惠票皆为当日有效。

  • 5.门票不记名，不挂失，公园对假票、过期票不负责任。

  • 6.门票对公园内需另行付费的项目无效；门票一经售出，非法定事由，不予办理退换票手续，使用一次有效，出园后入园需另行购票。

  • 7.为确保安全，公园可能因遇恶劣天气、特殊活动、游客过多、设备故障检修等原因调整部分设备和表演的时间或闭园，以当天公告为准。
