• 儿童翻江倒海

    专为儿童打造的高挑战滑道，让孩子们都能享受征服水上世界的乐趣，在专属的儿童天地里收获无限快乐！

  • 儿童迷你滑道

    色彩亮丽、五彩缤纷的迷你欢乐滑道专为儿童打造，酣畅淋漓地彻底感受亲水的无穷乐趣！

  • 儿童竞速滑道

    跟着水花一同从五彩滑道上滑落水底，劈波斩浪激起漫天水花，还等什么，小朋友们快来参加比赛吧。

  • 欢乐水屋

    让孩子们在色彩鲜艳的欢乐水屋，开始奇趣的水上探险之旅吧！蝴蝶滑梯、青龙滑梯、鳄鱼滑梯，让孩子们尽情嬉戏疯狂，还有丰富的互动游戏，激发您全身的欢乐细胞，带您体验全新的水上乐趣！

  • 漩涡巨碗

    漩涡巨碗，急弯滑道与“巨碗”的完美组合，在密封的滑道中，一场演绎极速与心跳的戏水大戏即将上演。一阵在急弯滑道中感受速度与旋转之后，刹那间掉进 一口巨碗，如同掉进深海漩涡。

  • 疯狂海啸

    海啸来了！巨大的垂直回旋滑道会把您从最低点带到最高点，然后在滑道中做自由回旋下落，落差达到15米，请您屏住呼吸，准备接受挑战吧。

  • 蛇形穿越

    随着巨蟒的节奏左右摇晃、上下颠簸，伴随360°急速回旋，穿越层层水的阻力，震撼惊叫声穿跃整个乐园，高刺激性、高娱乐性、高速度的三重水上体验等着您来挑战！

  • 蛇形滑道

    想在巨蟒的身躯上展开冒险的快感吗，那就极速下滑吧。滑行中途的几个螺旋，让滑行速度越来越快，并造成人体的连续失重感，犹如在水上裸体滑雪一样，比飞机上遇到乱流还要刺激，不信你就试试看！

  • 极速鲨吻

    亚洲首座穿越海底世界及室内外的水磁过山车。运用最新高科技专项打造，首创室内外穿越式互动游乐体验，创新设计贯穿陆地与水底。高速疾驰体验心跳旋律，带给您自由飞翔的畅快感受；潜入水底世界与大鲨鱼群一“吻”而过，体验惊险水族，零距离感受海洋缤纷，进行一场浪漫的历险奇遇。多重震憾欢乐，给您带来炫酷无比的新鲜刺激！

  • 棕榈海滩

    纯净洁白的细沙和透明闪亮的海域，1400平米造浪池，让您顷刻间身临阳光海岸。你可以在棕榈树下放一把躺椅，充分享受阳光，亦或者光着脚丫在海滩上漫步，听着海浪的声音……最后棕榄海滩形成一道浪漫梦幻，而又充满遐想的亮丽风景线。

  • 懒人河

    宽4米长261米的室内漂流河。坐上梦幻伐船，开阔水道清澈蜿蜒，两岸繁花绿丛，让您置身最自然生态的漂流景观。沿途穿越炫丽大型水族箱，近距离观赏美妙水底世界，创新动感影音特效，多重体验感受，带您探索峡谷、山林、涵洞、花海等缤纷景点，享受一段浪漫温馨的欢乐时光！
