• 恺撒宫

    富丽堂皇的宫殿内正举办一场鸡尾酒聚会,"魔术调酒师"就是欢乐的制造者,无限的欢乐将从他手中那五颜六色的酒杯中展开……

  • 极限运动营

    直排轮、滑板、极限单车、攀岩……各种各样的极限运动让你目不暇接,同时你也能来一展身手,激发出自己的潜在能量,超越极限！

  • 森林剧场

    如果你崇尚环保健康的生活方式，如果你是热爱大自然的低碳族，那么就来羽落天堂吧，在森林剧场里看一场精彩的表演，与翻飞的鸟儿、盛开的花儿一起享受大自然的美好。这里将让你置身世外桃源，远离城市喧嚣。

  • 打码头

    在水岸边搭台兴建的剧场内,码头边的风云变幻的传奇故事正精彩上映,当年的场景完美再现,让你对故事中的曾经如痴如醉。
