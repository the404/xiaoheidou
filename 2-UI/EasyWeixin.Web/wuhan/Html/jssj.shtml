• 极速飞车

    飞车会像导弹一般弹射出去,弹射瞬间的加速度让你摆脱地心引力,每一个仰角急弯、每一次倒翻、每一下俯冲都让你纵情咆哮,飞驰电掣间满足你对速度的所有激情与渴望！

  • 蒙地卡罗赛道

    这是F1赛车摩纳哥大奖赛的试炼场！为所有赛车手进行驾驶技术的严格考验,只有最顶尖的车手才能在这条青绿色的赛道上获得胜利！

  • 霹雳战车

    准备好迎接一轮又一轮火花四溅的碰撞吗？登上霹雳战车一比高下吧,赛车手们要把自己的驾驶技术发挥到极致,在躲避敌人撞击的同时,配合队友将对手击垮！

  • 太阳神车

    这一台如大摆锤一般的庞大巨物,将带着你超越失重状态和加载状态下的极限！随着圆盘的大幅摇摆,眼前的世界也颠倒成另一幅模样！
