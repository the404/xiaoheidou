选择你去上海欢乐谷的交通方式

  • 自驾
  • 公交
  • 步行

交通说明

欢乐谷位于上海市西南部的松江佘山国家旅游度假区内，距上海市中心区40公里，离虹桥国际机场25公里、浦东国际机场68公里。

  • 地铁：乘坐上海地铁九号线在佘山站下车，过天桥坐欢乐谷免费接驳车五分钟即至上海欢乐谷。
  • 公交：乘松江19路、松江19路区间、松重线在欢乐谷站下，步行5分钟即可至上海欢乐谷。
  • 自驾车：江苏：沪宁高速G2（A11）嘉松公路出口南下，至沈砖公路再西行百米。
    上海：沪杭高速G60（A8）松江出口，沿嘉松公路至沈砖公路再西行百米。
  • 停车场：上海欢乐谷配套佘山国家旅游渡假区停车场共有2000余个停车位。
  • 年卡办理路线：地铁九号线至佘山站下车，过街至欢乐谷接驳站搭乘公园班车，至公园正门口即可。
