• 《新上海滩风云》大型影视特技表演

    演出地点： 上海滩影视特技表演场
    以三十年代旧上海滩为背景，用影视特效、维亚、枪战等手段再现三十年代中共地下党人在上海滩与特务、黑帮，以及租界洋人周旋斗争的情景。游客在观赏的同时也可以参与其中，扮演其中的群众角色。

  • 《满江红》大型实景马战表演

    演出地点： 香格里马战场
    表演以我国南宋时期爱国名将岳飞的事迹为背景，采用马战实景表演的形式。现场将有百匹骏马、百名骑士奔腾厮杀，并运用万箭齐发、水爆、火爆等影视手段，逼真再现岳家军英勇善战、不屈不挠的恢宏场景。令观众在体验娱乐，感受震撼中走进岳家军，共同领略中华文化的神奇魅力。

  • 《奇幻之门》大型原创魔术剧

    演出地点：亚瑟宫剧场
    《奇幻之门》大型魔术剧是一场融合杂技、滑稽、舞蹈、魔术等多种艺术元素的大型原创魔术剧。

  • 《欢乐之旅》大型多媒体服饰歌舞秀
